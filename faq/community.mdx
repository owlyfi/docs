## Getting Help

### Support Tiers

| Channel | Response Time | Best For |
|---------|--------------|----------|
| **Discord** | Minutes - Hours | General questions, community help |
| **Email** | 24-48 hours | Account issues, private matters |
| **Documentation** | Instant | Self-service learning |

### Discord Channels

- `#general` - Community chat
- `#support` - Technical help
- `#announcements` - Official updates
- `#feedback` - Feature requests and suggestions
- `#traders` - Strategy discussions

## Report Issues

### Bug Reports

Found a bug? Report it:
1. Join Discord
2. Go to `#bug-reports` channel
3. Use the bug report template
4. Include screenshots and steps to reproduce

### Security Issues

For security vulnerabilities:
- Email: security@owly.fi
- **Do not** post publicly
- Bug bounty program available

<Note>
  Our team monitors all channels during business hours (UTC 9:00-18:00). Discord community members may also help outside these hours.
</Note>
