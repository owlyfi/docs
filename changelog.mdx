## v1.5.1 (2026-01-26)
### New
- Multi-Dimensional Ranking System: Launched a new discovery framework that replaces single-factor sorting with multi-dimensional rankings, enabling users to identify "smart money" across various metrics to build more robust portfolios. \
- Portfolio Strategy Control: Introduced portfolio-level Stop Loss and Take Profit (SL/TP) configurations, allowing users to manage risk across an entire portfolio rather than individual positions. \
- Hyperliquid Sub-accounts: Enabled full support for Hyperliquid sub-account creation, management, and API integration. \
- Smart Address Search: Implemented fuzzy search for trader addresses with keyboard navigation and UI feedback.

### Improvements
- Navigation Persistence: Discovery pages now automatically restore scroll positions and filter states for a smoother browsing experience. \
- ROI Accuracy: Refined ROI calculation logic and added informative tooltips for more precise performance tracking.

### Fixes
- System Limits: Adjusted minimum withdrawal amount to 10

## v1.4.0 (2026-01-14)
### New
- Manager Mode: You can now easily add and manage sub-accounts as a manager or owner.
- Spot Tracking: Added support for viewing spot holdings in trader profiles.
- Fee Transparency: A new tab is available to track your trading fees clearly.

### Improvements
- Faster Loading: Optimized data fetching to make account switching and balance updates much quicker.
- Mobile Experience: Improved chart layouts to ensure everything looks great on your phone.
- Smoother Browsing: The Discovery page now remembers where you scrolled and highlights your last click.

### Fixes
- Better Stats: Fixed the win rate formula to ensure break-even trades don't skew your performance.
- UI Cleanup: Fixed text display issues and corrected several terminology translations.
