Owly 上的每笔交易都记录在链上，确保完全透明和可追责。

## 为什么透明性重要

<AccordionGroup>
  <Accordion title="经过验证的历史记录" icon="check">
    所有业绩数据直接来自区块链记录，而非自我报告的指标。
  </Accordion>
  <Accordion title="无选择性展示" icon="chart-line">
    查看所有交易，而不仅仅是交易员想展示的盈利交易。
  </Accordion>
  <Accordion title="实时更新" icon="sync">
    业绩指标随着交易执行实时更新。
  </Accordion>
</AccordionGroup>

## 我们追踪的指标

<CardGroup cols={3}>
  <Card title="总盈亏" icon="dollar-sign">
    所有时间段的总利润/亏损。
  </Card>
  <Card title="投资回报率" icon="percent">
    投资回报百分比。
  </Card>
  <Card title="胜率" icon="trophy">
    盈利交易的百分比。
  </Card>
  <Card title="最大回撤" icon="arrow-down">
    最大峰值到谷底的跌幅。
  </Card>
  <Card title="夏普比率" icon="chart-bar">
    风险调整后的回报指标。
  </Card>
  <Card title="交易频率" icon="clock">
    每个周期的平均交易次数。
  </Card>
</CardGroup>

## 数据来源

| 数据类型 | 来源 | 验证方式 |
|-----------|--------|--------------|
| 交易历史 | Hyperliquid L1 | 区块链 |
| 仓位数据 | Hyperliquid API | 实时 |
| 业绩指标 | 链上计算 | 可验证 |
| 费用记录 | 智能合约 | 不可篡改 |

<Note>
  您可以通过在 Hyperliquid 的区块浏览器上查看任何交易员的地址来独立验证其业绩。
</Note>
