## 账户类型

理解 Owly 支持的三种跟单账户类型对于选择最适合您的方案至关重要。

### 1. API 钱包 (API Agent Wallet)
这是 Owly 推荐的高级用法。
- **机制**：利用 Hyperliquid 的 Agent Wallet API 进行交易。
- **性质**：**非托管**。您拥有完全的资金控制权。
- **特点**：每个子账户本质上是一个独立的 Hyperliquid 交易账户，通过 API 授权进行管理。
- **优势**：无需主账户拥有大额交易量即可创建，适合大多数用户。

### 2. Hyperliquid 子账户 (Hyperliquid Sub-account)
利用 Hyperliquid 原生的子账户功能直接导入。
- **机制**：直接关联您现有的 Hyperliquid 子账户。
- **性质**：**非托管**。
- **特点**：资金都在一个 Hyperliquid 主账户下操作转移。
- **优势**：资金管理统一。
- **劣势**：有一定门槛，Hyperliquid 要求主账户在 Hyperliquid 上有 **100,000 USDC** 的总交易量才能创建子账户。

### 3. Owly 托管账户 (Owly Managed)
- **机制**：Owly 为您创建并管理的账户。
- **性质**：**托管**。
- **特点**：操作极其简单，通过简单的点击即可生成。
- **劣势**：用户不直接掌握钱包私钥，安全性依赖于平台。
- **建议**：我们更推荐使用前两种非托管方案，以确保资产的绝对安全。

### 账户类型对比表

| 特性 | API 钱包 (API Agent) | Hyperliquid 子账户 | Owly 托管账户 |
|------|---------------------|-------------------|--------------|
| **类型** | 非托管 (推荐) | 非托管 | 托管 |
| **资金控制** | 用户完全控制 | 用户完全控制 | 平台管理 |
| **创建门槛** | 无交易量要求 | 需 Hyperliquid 交易量 (>100k) | 无 |
| **操作复杂度** | 中 | 中 | 低 |
| **适用人群** | 大多数用户 | 交易大户/专业交易员 | 新手/小额体验 |

## 关键指标

在选择交易员时，请参考以下指标。

### ROI (投资回报率)
`ROI = (当前价值 - 初始价值) / 初始价值 × 100%`
衡量策略的盈利能力。我们通常展示 7天、30天或总 ROI。

### 夏普比率 (Sharpe Ratio)
衡量每承担一单位风险所获得的超额回报。
- **> 1**：良好
- **> 2**：优秀
- **> 3**：卓越
夏普比率越高，说明在相同风险下获得的收益越高。

### 最大回撤 (Max Drawdown)
该策略历史上从最高点下跌的最大幅度。
- **意义**：代表了最坏情况下的潜在损失。回撤越小，策略越稳健。

### 胜率 (Win Rate)
`胜率 = 盈利交易数 / 总交易数`
- **注意**：高胜率不一定代表高收益（如果单笔亏损很大的话）。应结合盈亏比一起看。

### 波动率 (Volatility)
衡量收益的波动程度。波动率越低，收益曲线越平滑。

## 跟单系统逻辑

### 多维排名 (Ranking System)
Owly 摒弃了单一的 ROI 排名，采用多维度评分系统来寻找"聪明的钱"：
- **收益能力**：ROI, 绝对收益额
- **风险控制**：回撤控制, 夏普比率
- **稳定性**：交易频率, 资金容量
我们通过算法综合这些指标，为您推荐综合表现最优的交易员。

### 投资组合级止盈止损 (Portfolio SL/TP)
不同于传统的单笔订单止损，Owly 提供投资组合级别的风险控制。
- **原理**：系统监控整个子账户的净值。
- **触发**：当子账户总权益达到设定的止盈或止损线时，系统会自动平掉所有仓位并停止跟单。
- **优势**：有效防止单笔黑天鹅事件导致整个账户归零，保护您的本金。
