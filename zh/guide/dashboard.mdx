## 业绩图表

### 盈亏曲线

业绩图表显示不同时期的回报：
- **1天**：过去24小时
- **7天**：过去7天
- **30天**：过去30天
- **全部**：自账户创建以来

### 阅读图表

- **绿色区域**：累计盈利期
- **红色区域**：累计亏损期
- **曲线**：累计权益曲线

## 子账户卡片

每个子账户卡片显示：

| 字段 | 描述 |
|-------|-------------|
| **余额** | 账户中可用的 USDC |
| **权益** | 包含持仓的总价值 |
| **盈亏** | 选定时期的利润/亏损 |
| **ROI** | 投资回报率百分比 |
| **状态** | 活跃/空闲跟单交易状态 |

## 理解指标

### ROI（投资回报率）

```
ROI = (当前价值 - 初始价值) / 初始价值 × 100%
```

### 胜率

```
胜率 = 盈利交易数 / 总交易数 × 100%
```

### 最大回撤

投资组合价值从峰值到谷底的最大跌幅。

<Tip>
  关注风险调整后的回报而非绝对盈亏。一个50%回报、10%最大回撤的交易员通常比100%回报、50%回撤的交易员更可靠。
</Tip>
